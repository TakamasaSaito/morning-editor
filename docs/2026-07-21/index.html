GeminiだけがAndroidのシステム機能（音声起動・アプリ横断操作）を利用できるという現在の非対称構造が違法と認定され、ChatGPTやClaudeなど競合AIが同等の深度でAndroidに統合できる道が初めて開かれる。違反時はGoogleの年間世界売上高の最大10%の制裁金が科される可能性がある。

EUがGoogleにAI開放を強制命令

背景

欧州委員会は2026年1月27日にGoogleに対する2件の仕様特定手続きを開始し、6か月の法定期限内に拘束力ある決定を出した。現状ではAndroid上でChatGPTやClaudeをダウンロードしても「アプリ」にとどまり、Geminiが享受するウェイクワード起動・画面読み取り・アプリ内操作といったOSレベルの機能が使えない。EU内のAndroidユーザーの60%がこの制限を受けており、委員会はこれを競争上の不公正と判断した。EU一般裁判所は7月8日にApple案件で『逐次審査ルール』を確立し、Googleはこの決定への事前差し止め申請ができなくなった。

今後の展望

Googleは採択直後にKent Walkerグローバル担当社長名で声明を出し、プライバシーとセキュリティへの悪影響を訴えて決定後の法的争いを示唆している。検索データ共有（2027年1月）およびAndroid改修（2027年7月）という2段階のタイムラインが確定しており、Googleがこれを無視した場合は非遵守手続きと追加制裁が始まる。英国CMAはEUと異なる事例ごとのアプローチをとるため、英国市場での開放は遅れる見通し。

あなたへの影響

エンタープライズアーキテクトにとって直接的な設計変更が迫られる可能性がある。社内向けAIアシスタントやモバイルデバイス管理（MDM）の観点で、2027年以降のAndroid端末ではGemini以外のAIエージェントがシステム深部に統合される前提でアーキテクチャを再評価すべき局面だ。また、GoogleのSearch APIへの競合アクセスが正規化されれば、RAGパイプラインやエンタープライズ検索基盤の調達戦略にも影響が出る。

NVIDIAと日本、世界初の国家物理AIインフラ始動

背景

NVIDIAは5月にCosmos 3を発表済みで、今回のCosmos 3 Edgeはエッジデバイス上でリアルタイムのビジョン推論とロボット制御を行う40億パラメータの小型世界モデルだ。約1日で特定ロボット・車両・センサーへの適用が可能で、Jetson T2000/T3000を含むNVIDIAのエッジ製品上で動作する。日本のAI市場は2029年までに279億ドル規模に達すると見込まれており、マイクロソフトの100億ドル投資に続くNVIDIAの大規模コミットメントとなる。

今後の展望

Noetraのファクトリーは2027年4月に稼働開始予定。RIKENでもNVIDIA GB200搭載スーパーコンピューター2台が運用を開始しており、日本の製造業へのフィジカルAI浸透は2027年以降に本格化する見込みだ。

あなたへの影響

国内の製造・物流・ヘルスケア顧客を持つエンタープライズアーキテクトにとって、NVIDIAのCosmos/Isaacスタック上でデジタルツインや自律エージェントを設計する案件が急増する可能性がある。IT戦略立案の時間軸に2027年稼働のNoetraファクトリーを組み込むと、インフラ調達の優先順位が変わりうる。

骨太方針、日銀独立性を明記し本日閣議決定

背景

6月末に公表された原案には「強い経済の実現に向けて適切な金融政策運営が非常に重要」との文言が盛り込まれ、市場は日銀の利上げ牽制と解釈して長期金利が上昇した。政府は7月7日の第1次修正でも金利上昇を止められず、日銀法第3条への言及という第2次修正に追い込まれた。一方、消費税減税の結論時期については高市首相が今週中に判断するとされており、依然不透明だ。

今後の展望

日銀法第3条の明記は表面上の独立性担保だが、市場では「財政健全化」の文言が「財政の持続可能性」に置き換えられる点を引き続き注視している。10年金利が3%を突破するかどうかが次の焦点で、突破すれば再度の市場混乱が起きる可能性がある。

あなたへの影響

長期金利の動向はクラウドや大規模AIインフラへの国内設備投資コストに直結する。IT予算の資金調達コストや顧客企業の投資判断がタイト化するリスクを戦略立案に織り込む時期に来ている。

ウクライナ、400機超のドローンでモスクワを攻撃

背景

直前の週末、ロシアはキーウなどへの弾道ミサイル攻撃を実施し、少なくとも6人が死亡。ウォーシントン拠点のシンクタンク・戦争研究所（ISW）は「ロシアは7月だけで月間生産量を超える弾道ミサイルをウクライナに発射しており、在庫を取り崩している」と分析した。ウクライナ国内でもゼレンスキー大統領が国防相を更迭した閣僚再編が軍の世代間対立を露呈させ、政治的にも揺れている局面だ。

今後の展望

ゼレンスキーは「モスクワとサンクトペテルブルクに届く攻撃が続く」と宣言しており、ドローン攻撃のスケールアップが続く見通し。一方でロシアが弾道ミサイル在庫を消耗するペースで使い続けることは中長期的なロシアの戦力持続性に疑問を投げかける。

あなたへの影響

ロシア・ウクライナの継続的衝突は欧州エネルギー市場の不安定要因であり続ける。中東（米イラン衝突）との二重リスクが重なる中、エネルギーコストと物流リスクに敏感な顧客企業のIT投資計画に影響が及ぶ可能性がある。

Claude Sonnet 5がエージェントAI商用展開を牽引

背景

Stanfordの調査によれば、GPT-3.5レベルの推論コストは2022年末から2025年末にかけて280倍以上低下している。7月はClaude Sonnet 5の他、GitHub CopilotがオープンウェイトのコーディングモデルをAI初導入し、GoogleはVideo/画像ツールを追加した。アクセス制御面では本人確認・審査プレビュー・クレジット課金が普及し、強力なモデルほど使用条件が厳格化する傾向も強まっている。

今後の展望

9月以降はClaude Sonnet 5の価格が入力3ドル/出力15ドルへ上昇し、エージェント利用コストが上がる局面を迎える。アクセス条件の厳格化と推論コスト低下という二つの流れが共存する中、エンタープライズ向けには年間契約・プライベートデプロイの交渉余地が広がる可能性がある。

あなたへの影響

エンタープライズアーキテクトとしては、コーディングエージェント・ドキュメント処理・社内ナレッジ検索に対してClaude Sonnet 5系モデルの試験採用を8月末の価格変更前に開始し、TCOを実測するウィンドウが今まさに開いている。
