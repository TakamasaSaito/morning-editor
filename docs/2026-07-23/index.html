IBM株25%暴落、AIシフトの直撃弾

IBMは2026年第2四半期の速報値として売上高177億ドルを下回る172億ドル（前年比+1%）、調整後EPSは2.93ドルを発表し、市場予想を大幅に下回った。7月14日の事前公表後に株価は約25%急落し、同社発表によると1987年10月のブラックマンデー以来最大の単日下落となった。

IBMのCEO アービンド・クリシュナ氏は、企業顧客がAIインフラ向けサーバー・ストレージ・メモリの調達に資本支出を振り替えたことが主因と説明しており、AI投資拡大が既存ITサービス・ソフトウェア予算を食う「共食い構造」が鮮明になった。ServiceNow・Salesforceなどの関連株も5〜8%前後の連鎖安を記録したと複数メディアが報じており、1社の問題にとどまらない業界横断的シグナルとして受け止められている。

IBM株25%暴落、AIシフトの直撃弾

背景

IBMのインフラ部門売上は四半期で7%減少し、メインフレーム（Zシリーズ）の販売鈍化とトランザクション処理ソフト需要の低下が響いた。クリシュナ氏は「顧客が6月末にサプライ制約と価格上昇を見越して駆け込み購入に走り、複数の大口契約が四半期内に締結されなかった」と説明した。一方、ソフトウェア部門は5%増収を維持し、IBMが発表した累計AI受注残は120億ドルを超えた。7月22日（日本時間23日朝）に最終決算を公表予定で、速報値との差異が生じる可能性もある。

今後の展望

クリシュナ氏はAIを成長エンジンと位置付けているが、複数の報道はレガシービジネスからAI主導型収益構造への転換が依然として不均一であると指摘している。同様の構造的圧力は他のレガシー系ITベンダーにも波及する可能性があり、エンタープライズIT支出の優先順位再編が今後数四半期で加速するとの見方が出ている。

あなたへの影響

エンタープライズアーキテクトとしては、顧客・社内の予算承認プロセスにおいてAIインフラ投資が既存の保守・ライセンス費用を圧迫するシナリオへの対応が問われる局面になりつつある可能性がある。IBMコンサルティングや同社製品を基幹に組み込んだアーキテクチャを持つ組織では、ベンダーのロードマップ変更リスクをIT戦略に織り込む必要が出てくるかもしれない。AI対応のモダナイゼーション案件を提案する際、このような「AIが既存予算を侵食する」という構造を経営層に示す根拠として活用できるだろう。

ゼレンスキー、軍総司令官を電撃解任

背景

シルスキー氏（60歳）はモスクワの軍学校出身でソ連式の指揮体制を代表する存在と見られており、ドローン活用などテクノロジー主導の改革を推進したフェドロフ国防相と対立していたと複数メディアが報じている。フェドロフ氏の突然の解任が抗議運動を引き起こし、シルスキー解任へとつながった。後任のドラパティ氏はクリヴィリフやヘルソン方面での戦果で知られ、フェドロフ氏の盟友と位置付けられている。

今後の展望

ゼレンスキー氏はドラパティ任命と同時に参謀本部の再編も表明しており、ドローン・データ活用を軸とした近代化路線への転換が本格化するとみられる。フェドロフ氏に別ポストを提示したと報じられており、同氏の復帰を求める抗議運動が完全に収束するかは不透明な状況が続いている。

あなたへの影響

ウクライナの軍指導部が「テクノロジー主導型」へ転換する動きは、欧米の防衛テック企業への投資やAI軍事利用の議論を加速させる可能性がある。IT戦略の観点では、防衛分野でのAI・ドローン技術が民間エンタープライズIT調達にも波及する事例として注目に値するかもしれない。

MS・Mistral、欧州AIデータセンター共同増強

背景

EUは7月20日にGoogleに対してAndroid上でライバルAIを開放するよう命令（既報）するなど、AIプラットフォームの競争環境整備を進めている。こうした規制圧力と旺盛なAI需要を背景に、欧州域内でのインフラ投資競争が激化している。Mistralはフランス発のオープンソース志向LLM企業として欧州AI自立の象徴的存在であり、Microsoftとの提携強化は資本調達と技術力の双方を狙った戦略と見られている。

今後の展望

Vera Rubin世代のGPU基盤が欧州で整備されることで、金融・医療・行政など規制の厳しいセクターへのAI導入ハードルが下がると予想される。今後は他のクラウドプロバイダーやAI企業も欧州拠点強化を急ぐ可能性がある。

あなたへの影響

欧州拠点の顧客・パートナーを持つ組織や、グローバルなエンタープライズアーキテクチャを設計する立場では、AIインフラのリージョン分散戦略を今から組み込んでおく価値が高まっている可能性がある。コンプライアンス要件ごとに異なるモデルやスタックを使い分ける「マルチリージョンAI」設計の需要が本格化するかもしれない。

ロビー活動の優先議題としてサイバーセキュリティ・著作権・クラウドコンピューティング・国防調達が挙げられており、AIが政策のあらゆる側面に絡む段階に入ったことを示している。Metaは同期間に599万ドルでテック最大のロビイストとなったが前四半期比では15%減となっており、AI特化企業との比較で対照的な動きとなっている。

TSMC、米国投資を2650億ドルに拡大

背景

TSMCはアリゾナ州などへの工場建設を進めており、今回の投資拡大発表はその延長線上にある。米国の対中半導体規制強化を背景に、主要顧客であるAppleやNvidiaも米国内生産の確保を強く求めている。インドも7月22日に新たな半導体産業支援策「Semicon 2.0」を閣議承認しており、半導体製造をめぐる各国の誘致競争が激化している。

今後の展望

米国内工場の本格稼働には数年を要するが、AIインフラ需要が続く限り生産能力確保は最優先課題であり続ける。地政学リスクをヘッジしつつ先端プロセスを確保できるかどうかが、クラウド各社の競争力を左右する可能性がある。

あなたへの影響

エンタープライズアーキテクトとして、AIインフラを自社構築するか主要クラウドに委ねるかの判断において、半導体の地産地消シフトがクラウドの価格・可用性にどう影響するかをシナリオに入れておく価値が高まっているかもしれない。

日本の延長国会、会期末に向け攻防

背景

あなたへの影響
