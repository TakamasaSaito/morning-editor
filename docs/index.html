米イラン報復合戦、9日連続攻撃に

背景

2026年2月、米軍とイスラエル軍がイランへの攻撃を開始し、最高指導者ハメネイ師が死亡した。イランは反撃として中東の米関連施設を攻撃し、その後周辺アラブ諸国へも被害が拡大した。6月17日にトランプ大統領とイランのペゼシュキアン大統領が戦闘終結を定めた覚書に署名したが、その後も両軍の攻撃の応酬が続いている。内閣府は7月17日の報告書で、中東情勢の緊迫が世界的インフレを再燃させ、「実質的な購買力の低下や消費者マインドの悪化につながった」と指摘している。

今後の展望

米国では11月の中間選挙を控え、トランプ政権はホルムズ海峡の開放を最優先課題と位置づけているとの報道がある。イランが「封鎖カード」を手中にしている状況が続けば、エネルギー市場の不安定化が長期化する可能性がある。NYタイムズは米国が中東への軍用機追加派遣を検討していると報じており、さらなる攻撃拡大の可能性も否定できない。

あなたへの影響

ホルムズ海峡の緊張が続くことで、エネルギーコストの上昇がIT投資予算やクラウドインフラのコスト試算に影響を与える可能性がある。エンタープライズアーキテクトとしてはIT戦略の中長期コスト前提を見直す局面が来るかもしれない。また、日本関係船舶のホルムズ通過リスクが続く中、サプライチェーン依存度の高い製造業クライアントのリスクアセスメントが求められる場面も出てくるだろう。

スペインW杯制覇、4大会ぶり2回目

背景

今大会はアメリカ・カナダ・メキシコの3か国による史上初の共同開催で、出場枠が従来の32か国から48か国に拡大されて計104試合が実施された。スペインは19歳のパウ・クバルシとラミン・ヤマルを擁する若いチームで決勝まで勝ち上がり、アルゼンチンは後半アディショナルタイムにエンソ・フェルナンデスが退場となるなど苦しい展開の中で延長に突入した。

あなたへの影響

ビジネスの観点では、W杯後のスポーツテックや放映権・広告市場の動向がエンターテインメント・メディア系クライアントの戦略に影響する可能性がある。

背景

FRONTiaプロジェクトの背景には、政府が2026年3月に公表した「AIロボティクス戦略」がある。同戦略は2040年までのロボット世界市場を約60兆円規模と見通し、日本が世界シェア3割超（約20兆円）を獲得する目標を掲げている。Noetraが構築するAIファクトリーにはNVIDIAのVera Rubin GPU 27,500基以上が導入される予定とNVIDIAが発表している。

最低賃金、地域格差13年連続200円超

背景

日本では毎年夏に最低賃金審議会が改定額を審議する。近年は物価上昇を背景に引き上げ幅が拡大傾向にあるが、東京と最低水準の県の格差は依然として200円超を維持している。政府の「骨太の方針」でも賃上げが重点施策に位置づけられているが、地方の中小企業にとっては経営圧迫要因ともなっている。

今後の展望

2026年度の最低賃金改定は秋に答申が出る予定であり、物価動向や景気判断を踏まえた引き上げ幅が焦点となる。地方への人材分散を促す政策とセットでの議論が進む可能性がある。

あなたへの影響

IT・システム開発のオフショア・ニアショア拠点選定において、地方拠点の人件費優位性が今後縮小していく可能性があり、コスト試算の見直しが必要になるかもしれない。
